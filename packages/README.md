# Shared Packages Directory

This directory contains packages shared across client-server boundaries or between multiple applications in the monorepo:

1. **[core-types](file:///Users/prateeksharma/Developer/retriever/packages/core-types/):** Contains JSON schemas and static type definitions shared across JS/Python modules.
2. **[sdk-js](file:///Users/prateeksharma/Developer/retriever/packages/sdk-js/):** TypeScript/JavaScript client library for Retriever API integration.
3. **[sdk-python](file:///Users/prateeksharma/Developer/retriever/packages/sdk-python/):** Python client library for Retriever API integration.
